# Dossier Médias pour les Injections

Ce dossier contient les fichiers médias (images, vidéos) utilisés dans les injections de simulation.

## Structure suggérée :

```
media/
├── images/           # Images pour les injections
├── videos/           # Vidéos pour les injections
└── documents/        # Documents et autres fichiers
```

## Formats supportés :

### Images
- JPEG/JPG
- PNG
- GIF
- WebP
- SVG

### Vidéos
- MP4
- WebM
- MOV

## Utilisation

Les fichiers placés ici seront accessibles via l'URL :
`/media/[nom-du-fichier]`

Par exemple : `/media/images/injection1.jpg`
