## 🎮 MODULE 1: SIMULATION

**Quand l'utiliser ?**

- Créer ou lancer une simulation de crise
- Participer à un exercice
- Gérer des scénarios et injections

**Parcours type :**

1. Dashboard → 🎮 Simulation
2. Cliquer sur "Créer simulation"
3. Ajouter des scénarios et injections
4. Lancer la simulation

---

## 🎓 MODULE 2: INSTRUCTEUR

**Quand l'utiliser ?**

- Vous êtes formateur/animateur
- Vous gérez des équipes pendant une simulation
- Vous surveillez des incidents en temps réel

**Parcours type :**

1. Dashboard → 🎓 Instructeur
2. Vue Instructeur → Voir le statut global
3. Gérer les équipes et assigner des tâches
4. Consulter les incidents
5. Générer un rapport de session

---

## 📊 MODULE 3: BIA (Analyse d'Impact)

**Quand l'utiliser ?**

- Analyser la criticité des processus métier
- Générer des rapports de continuité d'activité
- Évaluer les risques et la conformité

**Parcours type :**

1. Dashboard → 📊 BIA
2. Dashboard BIA → Vue d'ensemble
3. Nouveau processus → Remplir le formulaire BIA
4. Rapports BIA → Générer un rapport PDF/DOCX
5. Usines → Analyse consolidée par site

---

## 📚 MODULE 4: WORKSHOP

**Quand l'utiliser ?**

- Organiser des formations
- Créer des plans d'action
- Gérer des événements de formation

**Parcours type :**

1. Dashboard → 📚 Workshop
2. Formations → Créer un module
3. Plans d'action → Définir les actions post-formation
4. Événements → Planifier une session
5. Notifications → Informer les participants

---

## 💡 Astuces Rapides

### Navigation Rapide

- **Dashboard = Point Central** : Toujours accessible pour revenir au tableau de bord
- **Icônes Colorées** : Chaque module a son émoji distinctif
- **Sous-menus** : Cliquer sur un module pour voir toutes les options

### Par Profil Utilisateur

#### 👨‍💼 Vous êtes Manager/Responsable BIA ?

➡️ Utilisez principalement : **📊 BIA** + **📚 Workshop**

#### 👨‍🏫 Vous êtes Instructeur/Formateur ?

➡️ Utilisez principalement : **🎓 Instructeur** + **🎮 Simulation**

#### 👨‍💻 Vous êtes Participant ?

➡️ Utilisez principalement : **🎮 Simulation** → Mode Participant

#### 🛡️ Vous êtes Admin ?

➡️ Accès complet + **🛡️ Administration**

## 🎨 Personnalisation

### Thème Clair/Sombre

- Allez dans **👤 Profil & Compte** → **Paramètres**
- Changez le thème selon vos préférences

### Notifications

- **📚 Workshop** → **Notifications**
- Configurez vos préférences de notification

---

## 🆘 Besoin d'Aide ?

### Problème d'Accès

- Vérifiez votre rôle (USER/ADMIN)
- Certains modules nécessitent des permissions spécifiques

### Page Introuvable

- Retournez au **Dashboard**
- Utilisez la navigation latérale pour accéder au module

### Questions Techniques

- Contactez **🛡️ Administration** → **Email Diagnostics**
- Ou consultez la documentation complète
